🧠 Problem Statement: Smart Notification System

You are building a Notification System for a company.

The system should:

Send notifications via:

Email

SMS

Push Notification

Support different message types:

Marketing message

Transactional message

Alert message

Allow adding new notification types (e.g., WhatsApp) in future

Allow adding new message types without modifying existing code

Log every notification sent

⚠️ Constraint:
The system must strictly follow SOLID principles.
